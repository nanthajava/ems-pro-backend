# Employee Management System (EMS-Pro)

A production-ready Employee Management System built using Spring Boot.

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs
- Lombok
- Spring Security

## Features

- Create Employee
- Get Employee
- Update Employee
- Delete Employee
- RESTful APIs
- MySQL Database
- Clean Architecture

## Project Structure

```
controller
service
repository
entity
dto
security
config
exception
util
```

## API Endpoints

POST /api/employees

GET /api/employees

GET /api/employees/{id}

PUT /api/employees/{id}

DELETE /api/employees/{id}
